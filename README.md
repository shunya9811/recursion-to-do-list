# To Do List

コンピュータサイエンス学習プラットフォーム Recursion のvueコースの課題として作成したプロジェクト「To Do List」です

## プロジェクトの概要

簡単な To Do List アプリです。テーマ設定ができ、そのテーマごとのタスクを追加できる形となっています。
